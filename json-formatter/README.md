# JSON Formatter

Format, validate, and minify JSON in your browser.

**[Try it live](https://json.evey.cc)**

- Syntax highlighting
- Error detection with line numbers
- Minify / prettify toggle
- Copy to clipboard
- Works offline (PWA)

## Tech

Single HTML file. No frameworks, no build step, no tracking.
All processing happens in your browser — nothing leaves your machine.
